## how to use

```html/javascript
<body>
    <div class="container">
        <canvas height="300px" width="300px" id="clock"></canvas>
    </div>
    <script type="text/javascript" src="clock.js"></script>
    <script>
        drawClock("clock");
    </script>
</body>
```

## How to change the clock's size

change <canvas> height and width

_notice:the height must equal to width_
